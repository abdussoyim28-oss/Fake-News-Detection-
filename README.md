# 📰 Fake News Detection using Machine Learning

## 📌 Project Overview

This project focuses on detecting fake news articles using Natural Language Processing (NLP) and Machine Learning techniques. The model classifies news as **real or fake** based on textual content, helping to combat misinformation.

---

## 🚀 Features

• Classifies news articles as **Fake or Real**
• Uses **TF-IDF Vectorization** for text representation
• Implements **Logistic Regression** for classification
• Trained on a dataset of **20,000+ news articles**
• Achieves high accuracy and strong performance metrics

---

## 🛠️ Technologies Used

* Python
* scikit-learn
* NumPy
* Pandas
* NLP (Natural Language Processing)
* TF-IDF Vectorizer

---

## 📊 Model Details

* Algorithm: Logistic Regression
* Text Processing: TF-IDF Vectorization
* Dataset Size: 20,000+ articles

### 🔍 Performance Metrics

* Accuracy: **94%**
* F1-Score: **0.94**

---

## 🔄 Workflow

1. Data Collection
2. Data Cleaning & Preprocessing

   * Lowercasing
   * Removing stopwords
   * Tokenization
3. Feature Extraction using TF-IDF
4. Model Training (Logistic Regression)
5. Model Evaluation (Accuracy, Precision, Recall, F1-score)
